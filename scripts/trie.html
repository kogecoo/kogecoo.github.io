What's this?

Visualization demo for help your understanding of Trie data structure.

Description

The Trie, one of a data structure for string strage which enables us to quick access to an entry.
We can understand the trie as only key-value stores, key is a string and value is any.
In above case, the string key is understood as the chain of the sub key which determined by the letter and its place in the string.
So, it can be naturally understood that Tries can save the space in case that trie has many similar strings.
