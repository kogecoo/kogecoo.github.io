Dancing Sudoku - The Sudoku solver using Dancing Links algorithm

What's this?

Just a simple Sudoku solver. If you want to cheating Sudoku puzzle, use this.

How to use

  1. You wish you could solve the Sudoku with no effort.
  2. Click the cell.
  3. Then, show up the text input form.
  4. Fill it by number from 1 to 9.
  5. Click the button "solve"


*Note : If there are multiple candidate of solution, the solver shows the one which is first found solution by internal algorithm.

Description

Sudoku[1], one of the most popular puzzle is the important material for study of the algorithms.
Solving the Sudoku is known as the problem categorized as NP-complete, so we have to use a brute-force storategy to solve it.

The Dancing Links algorithm[2] is introduced by Prof. Knuth as effecient backtracking algorithm.
This algorithm is easy to construct the backtraking solver for Exact Cover problem like Sudoku, 8 queens problem.[3]

This application is simple Sudoku Solver, using Dancing Links algorithm, and written by JavaScript.
This is not serverside application but just a client application. That is, all computation runs user's js interpreter ( or compiler ).

Motivation of using JavaScript is JavaScript engine of the recent webbrowsers is surprisingly quick.
So author thought that relatively heavy computation like this Sudoku solver will easy to run. And it proved.
