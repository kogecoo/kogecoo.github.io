What's this?

Simple demo for the Three-Way Quick Sort by JavaScript

Description

The 3-way quick sort is a variation of the quicksort.
Standard quicksort we know well uses a single pivot, but the 3-way quick sort uses two.

Module

Three Way Quick Sort module for JavaScript is available.
